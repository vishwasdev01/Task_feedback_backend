# Mini Feed App - Backend

This is the backend for the **Mini Feed App** — a simple feed where users can:
- View posts
- Create posts
- Like / Dislike posts

Data is stored **in-memory** — restarting the server will reset all posts.

---

## 📦 Tech Stack
- Node.js
- Express.js
- CORS
- In-memory storage (no database)
